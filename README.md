# java-test-0001-final-14525-yash
Final Project Assignment - This repository contains the complete final project code and documentation.
This project implements a Java program to print a binary triangle pattern using nested loops.
The values alternate between 1 and 0 in each row.

Pattern Output

1
01
101
0101
10101
010101

Logic Used

* The program uses two loops:
  * Outer loop → controls number of rows
  * Inner loop → prints elements in each row
* Odd rows start with 1
* Even rows start with 0
* Values toggle using:

value = 1 - value;

Complexity

* Time Complexity: "O(n²)"
* Space Complexity: "O(1)"

Programming Language

* Java
